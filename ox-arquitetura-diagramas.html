04 State Machine de Fase (Crítico para Pagamento)

Pagamento só é liberado quando o estado atinge "validated". Regra inviolável.

stateDiagram-v2 [*] --> pending pending --> in_progress: Worker inicia fase in_progress --> evidence_uploaded: 📸 Upload fotos/vídeo evidence_uploaded --> under_review: Cliente notificado under_review --> validated: ✅ Cliente aprova under_review --> rejected: ❌ Cliente rejeita rejected --> in_progress: Worker refaz validated --> [*]: 💰 Pagamento liberado note right of validated SOMENTE neste estado o pagamento é processado end note
